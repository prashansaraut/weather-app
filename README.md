# weather-app
A responsive weather application built using HTML, CSS, and JavaScript that fetches real-time weather data using a weather API.

## Features
- Search weather by city
- Real-time temperature
- Weather condition
- Humidity
- Wind speed
- Responsive design
- Clean UI

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Weather API
